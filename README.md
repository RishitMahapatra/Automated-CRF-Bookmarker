# Automated-CRF-Bookmarker
CRF Auto-Bookmarker: Desktop app to auto-create hierarchical PDF bookmarks for CRF documents—useful for CRF bookmarking in clinical trials (R&amp;D). Uses CustomTkinter and a PyMuPDF engine to detect “Folder:”/“Form:” headers, build Folder → Form TOC, display progress, and export a bookmarked PDF. Includes in-memory API and adjustable scan height.
